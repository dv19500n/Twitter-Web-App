# Twitter-Web-App
Full-stack Twitter-inspired social media web app built with Flask, JavaScript, DynamoDB, and S3.
### Features
- User signup/login
- Session authentication
- Create/Delete posts
- Reply system
- Profile pages
- Profile picture uploads
- Dynamic feed updates
- AWS DynamoDB integration
- AWS S3 image storage
### Technologies Used
- Python
- Flask
- JavaScript
- HTML/CSS
- AWS DynamoDB
- AWS S3
